# PyTuber
> To Download Youtube Videos Using Python ( Gui )

## Getting Start
   - Download `PyTuber`
   - Extract The Downloaded Rar
   - cd `PyTuber`
   - `pip install requirements.txt`
   - python PyTuber.py
